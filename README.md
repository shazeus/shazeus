### About

I’m Shazeus. I build Minecraft clients, terminal tools, and small projects around things I actually use.

My background is the Minecraft anarchy scene: CPVP, base hunting, utility clients, client setups, and server moments. I’m not an old 2b2t player; I come from the wider anarchy community.
